Experiment 6

Write XSLT Style-Sheet to Convert XML Document to HTML using XSL Template

Objective

To provide students in-depth theoretical base of web programming using XML.

Theory

XSLT (Extensible Stylesheet Language Transformations) is used to transform XML documents into other formats like HTML.

  • XSLT stands for XSL Transformations.
  • XSLT transforms an XML document into another format.
  • XSLT uses XPath to navigate XML documents.

XSLT allows adding/removing elements, sorting, and conditional transformations.

Example

XML Document

<?xml version="1.0"?>
<?xml-stylesheet type="text/xsl" href="p1.xsl"?>
<class>
    <student roll="1">
        <name>Ken</name>
        <age>26</age>
        <address>Pune</address>
        <subject>Maths</subject>
    </student>
    <student roll="2">
        <name>Boris</name>
        <age>23</age>
        <address>Kothrud</address>
    </student>
</class>

XSLT Stylesheet

<?xml version="1.0" encoding="ISO-8859-1"?>
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
    <xsl:template match="/">
        <xsl:for-each select="class">
            <xsl:apply-templates select="student" />
        </xsl:for-each>
    </xsl:template>
    <xsl:template match="student">
        <p>Roll: <xsl:value-of select="@roll" /></p>
        <p><b>Name: <xsl:value-of select="name" /></b></p>
        <p><i>Age: <xsl:value-of select="age" /></i></p>
        <p><u>Subject: <xsl:value-of select="subject" /></u></p>
    </xsl:template>
</xsl:stylesheet>

Conclusion

XSLT is a transformation language that converts XML documents into HTML.
