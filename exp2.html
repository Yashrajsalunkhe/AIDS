Experiment No: 2

Title: Programs based on typography and background properties of CSS, animation effect by using the transition feature of CSS.

CSS Transitions

CSS transitions allow you to change property values smoothly, over a given duration.

Example

        div {
          width: 100px;
          height: 100px;
          background: red;
          transition: width 2s;
        }
        div:hover {
          width: 300px;
        }

Transition + Transform

        div {
          width: 100px;
          height: 100px;
          background: red;
          transition: width 2s, height 2s, transform 2s;
        }
        div:hover {
          width: 300px;
          height: 300px;
          transform: rotate(180deg);
        }

Result:

The programs to demonstrate CSS transition properties were executed successfully.
