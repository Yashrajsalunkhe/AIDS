Experiment No: 1

Title: To Implement Different types of basic HTML Tag using HTML5 version.

Aim:

To Implement Different types of basic HTML Tag using HTML5 version.

Theory:

There is a list of newly included tags in HTML 5. These HTML 5 tags (elements) provide a better document structure.

List of HTML 5 Tags

  • <article> - Defines an independent piece of content.
  • <aside> - Specifies content slightly related to the main content.
  • <audio> - Plays audio files.
  • <canvas> - Used to draw graphics.
  • <footer> - Defines a footer section.
  • <header> - Defines a header section.
  • <main> - Defines the main content.
  • <mark> - Highlights text.
  • <nav> - Defines navigation links.
  • <section> - Defines a standalone section.
  • <summary> - Summary for a <details> element.
  • <video> - Plays video files.

Code Examples:

1. <article> Tag

        <article>
            <h2>EDUCBA</h2>
            <p>EDUCBA is a leading global provider of skill-based education.</p>
        </article>

2. <figure> Tag

        <figure>
            <img src="educba.png" alt="EDUCBA Logo"/>
        </figure>

3. <header> Tag

        <header>
            <h1>Header One</h1>
            <h2>Header Two</h2>
        </header>
